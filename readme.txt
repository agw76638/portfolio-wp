# 🎨 My Portfolio WordPress Theme

This is a custom WordPress theme I built for my personal portfolio site [ahndev.com](https://ahndev.com), designed to showcase my web development and freelance projects.

## 📌 Features

- Built on top of the Twenty Twenty-One theme
- Fully responsive and accessible
- Custom hero section, about section, contact form
- Project showcase using custom blocks and plugins
- Dark mode friendly with improved contrast styling

## 🚀 Live Preview

Visit the live site here: [https://ahndev.com](https://ahndev.com)

## 🛠️ Technologies Used

- WordPress block editor (Gutenberg)
- HTML, CSS, JavaScript
- Theme customization via `functions.php`
- Custom CSS for layout and responsiveness
